About This Chart

An interactive 3D stellar navigation chart inspired by the in-ship computer display from Project Hail Mary.

Star positions are computed from real astrometric data (right ascension, declination, parallax). The coordinate system is Sol-centred equatorial J2000, with the celestial equatorial plane as the reference grid.

The red dashed trajectory represents the route of the Hail Mary spacecraft from Sol to Tau Ceti (3.65 pc / 11.9 ly).
